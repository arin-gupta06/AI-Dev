# AI Development Learning Repository

A comprehensive, structured learning project covering Data Science fundamentals, Mathematics for AI, and Pythonic coding practices.

---

## 📁 Project Structure

### 1. **Data Science Essentials**
Core data analysis, manipulation, and exploration skills.

#### 📊 Numpy/
- **Part1.py** – NumPy array basics, creation, indexing
- **Part2.py** – NumPy operations and mathematical functions
- **Part3.py** – NumPy broadcasting and advanced operations
- **Part4.py** – NumPy linear algebra and matrix operations
- **Learnings.md** – Key NumPy concepts and takeaways
- **OneShot/** – Jupyter notebooks for quick reference
  - part_01.ipynb – Basic NumPy refresher
  - part_02.ipynb – Advanced NumPy patterns

#### 🐼 Pandas/
- **Part1.py** – DataFrame creation, loading, basic operations
- **Part2&Practice.py** – Indexing, selection, filtering
- **Part3&Practice.py** – Groupby, aggregation, sorting
- **Part4&Practice.py** – Merging, joining, reshaping data
- **Statistical_Analysis_Guide.py** – Descriptive statistics, distribution analysis
- **Statistical_Analysis_Practice.py** – Practice problems
- **Statistical_Analysis_Solutions.py** – Worked solutions
- **Pandas_Complete_Master_Guide.md** – Comprehensive reference guide
- **Learnings.md** – Key Pandas concepts
- **Data files:**
  - Coffee_Stats.csv – Coffee brand sales by month
  - student_performance.csv – Student scores by subject and gender
  - index_1.csv – Sample indexed data

#### 📈 Matplotlib/
- **Part1.py** – Basic plotting, figures, axes, line/bar/scatter plots

#### 🔬 Practice/
Real-world EDA and data analysis projects.

- **Combined.py** – Student performance EDA (gender comparison, statistical insights)
- **Combined_Part2.py** – Coffee sales EDA (month-wise correlation, brand pair gaps)
- **Numpy/** – Isolated NumPy exercises
  - BeginnerTextRevision.py – Beginner-level NumPy concepts
  - IntermediateRevision.py – Intermediate NumPy patterns
  - AdvanceRevision.py – Advanced NumPy techniques
  - Exercise1.py, Exercise2.py – Practice problems
- **Pandas/** – Isolated Pandas exercises
  - Ex1.py, Ex2.py, Ex3.py, Ex4.py – Pandas practice problems
  - dataset.py – Utility for loading sample data
  - iris.csv – Classic iris dataset

#### 📋 EDA_Data_Cleaning_7-Step_Playbook.md
**Core learning resource:**
A structured 7-step framework for exploratory data analysis:
1. Define objective
2. Create data dictionary
3. Run initial audit
4. Clean in small passes
5. Validate after cleaning
6. Save clean version
7. Write summary

Plus 4 deeper analysis steps: univariate, bivariate, correlation, outlier detection.

## 🎯 Learning Path

### For Beginners:
1. Start with **Data Science Essentials > Numpy > Part1.py**
2. Move to **Data Science Essentials > Pandas > Part1.py**
3. Follow the **EDA_Data_Cleaning_7-Step_Playbook.md**
4. Practice with **Practice > Pandas > Ex1.py, Ex2.py**

### For Intermediate:
1. Work through statistical analysis: **Pandas > Statistical_Analysis_Guide.py**
2. Complete EDA projects: **Practice > Combined.py, Combined_Part2.py**
3. Study **Maths > Linear Algebra > Day_01.py**
4. Explore advanced patterns: **Pythonic Code > ListComprehension > Part1.py**

### For Advanced:
1. Master advanced NumPy: **Practice > Numpy > AdvanceRevision.py**
2. Complete statistical solutions: **Pandas > Statistical_Analysis_Solutions.py**
3. Implement custom EDA frameworks
4. Build mini-projects in **Projects/**
